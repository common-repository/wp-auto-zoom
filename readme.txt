== Description ==

Plug-in convert automatically all links to images to a inner zoom window.

== Installation ==

1. Download the zip file and extract the contents.
2. Upload the folder to your plugins directory (wp-content/plugins/).
3. Activate the plugin through the 'plugins' page in WP.

== Using the plugin ==
4. All links in your post, that referes to images (like <a href="image.jpg">Test</a>), 
will display big image in your window.
